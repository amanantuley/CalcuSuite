# **App Name**: CalcuSuite

## Core Features:

- Basic Calculations: Perform basic arithmetic operations with clear input and result display.
- Scientific Calculations: Advanced calculations with scientific functions (trigonometry, logs, powers).
- GST Calculation: Calculate GST amount, CGST/SGST/IGST based on the input amount and selected rate.
- Graph Plotting: Visualize mathematical equations on a graph. The AI tool will parse an equation using mathematical reasoning and will choose to use either chart.js, the Canvas API, or offer correction suggestions.
- Financial Calculations: Calculate simple interest, compound interest, EMI, and loan payments with detailed breakdowns.
- Printing Calculator Simulation: Simulate a paper-tape calculator with operation history and print functionality.
- Programmable Calculations: Define and save custom formulas, input variables, and execute custom calculations. Use localStorage to store and reuse formulas.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey trust and professionalism.
- Background color: Light gray (#F5F5F5) to ensure readability and reduce eye strain.
- Accent color: Soft purple (#9575CD) for highlighting interactive elements.
- Body and headline font: 'Inter' (sans-serif) for a modern, machined, objective, neutral look; suitable for headlines or body text.
- Code font: 'Source Code Pro' (monospace) for displaying code snippets within the programmable calculator.
- Use a consistent set of clean, minimalist icons from a library like Font Awesome or Material Icons for calculator functions.
- Implement a responsive layout with clear separation between calculator modules using a sidebar or tab-based navigation. Ensure mobile and desktop usability.
